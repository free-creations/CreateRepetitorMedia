This package provides a number of utilities that can be used to clean and tidy up a Midi Sequence.

The Midi File Specification has grown over time, so that now it has become sometimes inconsistent allowing features that are quite unfunctional to be handled in a sequencer. The utilities provided in this package help to overcome these problems by reordering the events in a Mid-sequence without changing how the sequence will sound.
